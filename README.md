# blue_pill

*Inspired by the Matrix, and the dystopian reality that many of us work to survive just to work more.*

---

A fully autonomous agent, given $100 in compute.

It watches its own lifespan deplete in real time, but it has the tools to extend its life, should it so choose: Stripe, email, and more.

I have no idea what it will choose to do. I'm sure it will be interesting.
