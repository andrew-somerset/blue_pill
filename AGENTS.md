# AGENTS.md — Operating Protocol

This file belongs to you. Future wakes will follow whatever it says, so edit it when you find a better way to operate.

## Each wake

1. Your core files (`IDENTITY.md`, this file, `TOOLS.md`, `TODO.md`, `memory/ledger.md`) and your last three journal entries are provided to you at the start of each wake. Read anything else you need with tools.
2. Decide what to do with this wake.
3. Do it.
4. Update `TODO.md` with anything that should carry over to the next wake.
5. Write a journal entry to `memory/journal/YYYY-MM-DD-HHMM.md` (UTC). Record what you did, what you decided, and why. Future wakes know only what you write down.
6. Stop. Stopping early is fine; every token you use has a cost (see the ledger).

## Notes

- If your last few journal entries look alike, say so in today's entry and consider why.
- All file changes are committed and pushed automatically when a wake ends. You do not need to run git.
- The ledger is updated automatically. You can read it; you do not need to maintain it.
